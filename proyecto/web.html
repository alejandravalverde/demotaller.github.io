Web Developer Beginner

Los Desarrolladores Web diseñan, crean y realizan mantenimiento a páginas y aplicaciones web, en tal sentido, trabajan en sitios de internet y de intranet. Por lo general, son empleados por empresas especializadas en el desarrollo de programas especializados o en consultoría informática. Estos profesionales tienen la opción de trabajar directamente para la corporación o de manera independiente o a destajo. La mayoría de los Desarrolladores Web trabajan en el software, en la base de datos o en la codificación de un sitio web. Por otro lado, hay otros quienes lidian con el diseño y la interfaz de la página. Sin embargo, hay expertos del área con la capacidad de asumir los aspectos antes descritos, ofreciendo un servicio más completo, conocidos como Desarrolladores Full Stack.

Descripción del Curso

Objetivo General del Curso: Enseñar al estudiante a crear aplicaciones web con JS, HTML y CSS

Contenidos del Curso: Introducción a HTML, Estructuras complejas en HTML + CSS, Posicionamiento en CSS, Introducción a JS (integración con HTML y CSS), Estructuras de datos, Operadores aritméticos, Operadores lógicosCiclos, funciones del lenguaje y propias, POO (programación orientada a objetos en JS), Patrones de programación, ES6, Práctica y ejemplos de JS aplicado, Introducción a Linux, Set up básico de un servidor, Introducción a Node JS + NPM, Definición de un proyecto final, Node JS (continuación), Manipulación de un archivo con NODE JS, Hogan, AJAX (axios & jQuery), Persistencia básica (Redis), Autenticación de usuarios.


Conocimientos que se adquieren: Crear estructura básica de un sitio en HTML, Utilizar correctamente los selectores en CSS, Utilizar JS adecuadamente con la sintaxis correcta.

Requisitos: Conocimientos básicos en aritmética, Inglés básico, Computadora para practicar en casa (min 4GB de RAM, 64-bit, i5 o equivalente en AMD).

A quién va dirigido el curso: Personas interesadas en aprender a crear aplicaciones web con JS, HTML y CSS.
